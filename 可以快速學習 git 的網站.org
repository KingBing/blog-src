git 是非常強大的版本管理系統，但是要能夠良好的駕馭他，除了大量的實作經驗外，
有更好的學習輔助也是必須的，在此列出兩個學習 git 非常棒的網站。

* Learn Git Branching

http://pcottle.github.com/learnGitBranching/

這個網站除了外觀非常漂亮以外，也可以讓你在短時間內能夠更加理解 git。
